# Customer Intelligence System  
### (Segmentation + Churn Prediction + Recommendation)

## 📌 Overview
This project is an end-to-end data science solution that analyzes customer behavior to generate business insights. It combines customer segmentation, churn prediction, and recommendation systems into a single pipeline.

The system helps businesses:
- Identify high-value customers
- Detect customers likely to churn
- Recommend products to increase sales

---

## 🎯 Objectives
- Perform customer segmentation using RFM analysis and clustering
- Predict customer churn using machine learning models
- Build a recommendation system using association rules
- Provide actionable insights through data visualization

---

## 📊 Dataset
- Online retail / customer transaction dataset
- Features include:
  - Customer ID
  - Invoice Date
  - Product
  - Quantity
  - Price

---

## ⚙️ Project Workflow

1. Data Collection  
2. Data Cleaning  
3. Exploratory Data Analysis (EDA)  
4. Feature Engineering (RFM Analysis)  
5. Customer Segmentation (K-Means Clustering)  
6. Churn Prediction (Classification Models)  
7. Recommendation System (Apriori Algorithm)  
8. Visualization / Dashboard  

---

## 🧠 Techniques Used

- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature Engineering (RFM Model)  
- Clustering (K-Means)  
- Classification (Logistic Regression / Random Forest)  
- Association Rule Mining (Apriori)  

---

## 📈 Results

- Customers were grouped into meaningful segments such as:
  - High-value customers  
  - Loyal customers  
  - At-risk customers  

- Churn prediction model identifies customers likely to stop purchasing

- Recommendation system suggests products based on past purchase patterns

---

## 🛠️ Tech Stack

- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib, Seaborn  
- (Optional) Streamlit / Power BI  

---

## 📊 Key Insights

- A small percentage of customers contribute to the majority of revenue  
- Customers with low frequency and high recency are more likely to churn  
- Product bundling can increase sales through recommendations
